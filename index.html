Como Funcionam os Cupons de Desconto

Os cupons são códigos gerados para divulgar nas redes sociais, WhatsApp, panfletos, etc. Os cupons podem ser de uso geral ou de uso único.

No caso de cupons de uso geral, os clientes podem utilizá-los enquanto estiverem ativos na plataforma. Já os cupons de uso único só podem ser usados uma vez por cliente.
